# Cinematic Aspect Ratios
An addon for Blender which helps you setup the correct resolutions for many different aspect ratios and sizes.

## Supported Sizes:

* 5k
* 4k
* 4kHD
* 2k
* 1080p
* 720p

## Supported Aspect Ratios:

* 1.33:1 (4:3)
* 1.66:1 (5:3)
* 1.77:1 (16:9)
* 1.85:1
* 2:1
* 2.35:1
* 2.37:1 (RED Wide)
* 2.39:1 (Referred to as 2.40)
* 2.44
